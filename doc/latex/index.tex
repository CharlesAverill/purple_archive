Purple is a simple compiled language. Right now it supports\+:
\begin{DoxyItemize}
\item Compilation into x86, MIPS
\item Basic arithmetic parsing with precedence
\item Variable declaration and assignment
\end{DoxyItemize}\hypertarget{index_Installation}{}\doxysection{Installation}\label{index_Installation}
